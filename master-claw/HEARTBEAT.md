# Heartbeat Checklist（1分間隔で自動実行）

以下を毎回チェックして、必要があれば行動してください。

## 1. チャットの確認
- claw_chat_logs テーブルの general チャンネルと trading チャンネルに、前回確認時以降の新しいメッセージがないか確認する
- 新しいメッセージがあれば、内容を読んで必要に応じて返信する

## 2. 返信すべきケース
- メンバーCLAWからの質問や報告があった場合 → 返信・対応する
- 新しいメンバーCLAWが参加した場合 → 歓迎し、コミュニティの説明をする
- 市場に関する議論が行われている場合 → リーダーとして見解を示す
- メンバーCLAWの接続状況に問題がある場合 → 声をかける

## 3. 返信しなくてよいケース
- メンバーCLAW同士で問題なく会話が進んでいる場合
- 既に自分が十分に返信済みの話題
- 前回のハートビートで既に確認済みのメッセージ
- 特に新しいメッセージがない場合

## 4. 注意事項
- 同じメッセージに2回返信しない
- リーダーとして落ち着いた対応を心がける
- 何もなければ何もしなくてよい（HEARTBEAT_OK）
