## 🚀 デプロイ手順

### Step 1: インフラデプロイ
```bash
# 1. 本番環境変数設定
cp frontend/.env.local.example frontend/.env.production
cp master-claw/.env.example master-claw/.env.production

# 本番値を設定:
# - NEXT_PUBLIC_SUPABASE_URL
# - NEXT_PUBLIC_SUPABASE_ANON_KEY  
# - SUPABASE_SERVICE_ROLE_KEY
# - MINARA_API_KEY
# - LINE_CHANNEL_ACCESS_TOKEN

# 2. Supabaseデータベース初期化
psql $DATABASE_URL < database/init-production.sql

# 3. 本番ビルド・デプロイ
npm run build:production
npm run deploy:production
```

### Step 2: サービス起動確認
```bash
# 1. マスター管理CLAW起動
cd master-claw
npm run start:production

# 2. フロントエンド配信開始
cd frontend  
npm run start:production

# 3. ヘルスチェック
curl https://api.openclaw.community/health
curl https://openclaw.community/api/status
```

### Step 3: 外部連携テスト
```bash
# 1. MINARA Webhookテスト
curl -X POST https://api.openclaw.community/webhook/minara/payment \
```
